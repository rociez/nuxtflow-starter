# nuxtflow-starter

## 说明

- 基于 nuxt3 + vuetify3 + unocss 构建的脚手架
- 对 vuetify 的默认样式进行了定义
- 默认引入了 Unocss 的 presetWind4 预设和 transformerDirectives 预设
- 注意使用 Unocss 需要添加前缀 ***un-***

## 克隆到本地

If you prefer to do it manually with the cleaner git history

```bash
git clone https://github.com/kbysiec/vite-vanilla-ts-lib-starter.git
cd nuxtflow-starter
pnpm i
```

## 清单

当您使用此模板时，请更新以下内容：

- 删除 `.git` 目录，并运行 `git init` 来清理历史记录
- 在文件 `package.json` 修改 name 
- 删除 `README.md` 和 `CHANGELOG.md` 文件

祝您使用愉快 😊
